# Data Mining & Warehousing

This repository contains various data mining and warehousing projects and analysis scripts.

## Project Structure

- **[DMW05_Twitter](DMW05_Twitter/)**: Twitter sentiment analysis and visualization.
- **[P01_Scraping](P01_Scraping/)**: Web scraping for mobile data from Snapdeal.
- **[P02_NBA](P02_NBA/)**: Statistical analysis of NBA 2018-19 season data.
- **[P03_Employee](P03_Employee/)**: Human resources and employee data analysis.
- **[P06_Smoothing](P06_Smoothing/)**: Data preprocessing techniques including binning and smoothing.

## How to Run
Most scripts require `pandas`, `matplotlib`, `wordcloud`, `nltk`, and `textblob`.
To install dependencies:
```bash
pip install pandas matplotlib wordcloud nltk textblob emoji
```
Then run any script using:
```bash
python <script_name>.py
```
